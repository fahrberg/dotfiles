# Dependencies

List of required/recommended dependencies for new Arch installation:

```bash
sway                  # window manager
mako                  # notifications daemon
libnotify             # DNS implementation
waybar                # toolbar
ly                    # display manager
wl-clipboard          # clipoard utilities

kitty                 # terminal emulator
imagemagick           # for kitty to display images
fish                  # shell
starship              # fish prompt
ttf-fira-code         # terminal fonts

fzf                   # fuzzy search (used in launcher)
neovim                # text editor
yay                   # AUR helper (manual installation)
tig                   # git TUI
ripgrep               # cool grep

git                   # to get dotfiles
git-lfs               # to get dotfiles resources
nodejs                # for coc.nvim
npm                   # for coc.nvim
openssh               # git over ssh

alsa-utils            # alsamixer
pulseaudio            # sound
pulseaudio-alsa       # sound
pulseaudio-bluetooth  # sound
```
